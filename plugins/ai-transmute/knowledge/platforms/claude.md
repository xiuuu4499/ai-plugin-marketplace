Claude uses `.claude-plugin/plugin.json` and automatically discovers conventional component directories. Modern [skills](/components/skills.md) subsume most prompt-command use cases, while legacy [commands](/components/commands.md) remain supported. Plugin hooks receive Claude plugin root and data environment variables.
