# Maintain research

Read the job's source registry, knowledge log, approval record, and previous report. Add changed sources to the registry; never overwrite earlier evidence or decisions. Re-run duplicate, priority, conflict, and uncertainty analysis for the affected claims.

Use OKF maintain/curate practices on impacted concepts, update citations and indexes, and write a design-impact analysis that distinguishes unchanged conclusions, changed facts, and newly proposed changes. If a public interface changes materially, return it to `$review` for explicit human approval before implementation work resumes.
